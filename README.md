# Godot-First-Game

## Descriptionm
A simple but challenging platformer built using Godot. Player has to collect all the coins in the level and make it to the top as fast as they can.

## Features
Left and right movement<br />
Jumping<br />
Coin collection<br />
Timer<br />
Enemies<br />
Moving Platforms<br />
